港剧迷福音

Mar 15, 2019

最近发现一个超级厉害的应用：RSS Player

这个应用可以收看我最爱的港剧，好开心，在各台设备上都装上了 @_@

教程很多地方都有，很简单：

1.IOS/Android 下载 “RSS Player”

2.进入 “RSS Player” -> Channel: Add

name: tvb (any name)

link: http://allrss.se/dramas

3.点击 “RSS Player” -> Home, 点击刚刚创建的这个channel (e.g tvb)

搞定！
